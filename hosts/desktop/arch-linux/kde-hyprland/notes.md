# desktop / arch-linux / kde-hyprland

Machine **archpc** — Arch Linux desktop running both KDE Plasma and Hyprland.

- CPU: AMD Ryzen 7 9800X3D
- GPU: NVIDIA GeForce RTX 5070 Ti (discrete) + AMD Radeon (integrated)
- Display: Samsung LF27T850, 2560x1440 @ 60Hz, scale 1
- NVIDIA env + cursor + monitor live in `hypr-local.conf`.

Install: `./setup.sh desktop/arch-linux/kde-hyprland`
